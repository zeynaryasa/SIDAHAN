Panduan Instalasi

1. Buka folder di code editor masing-masing
2. Buat database pada phpmyadmin atau sejenisnya dengan nama si-pt_db
3. Lalu import file database yang berada pada folder "db" dengan nama "si-pt_db.sql"
4. Lakukan konfigurasi koneksi database pada "db/conn.php"
5. Point 4 Sesuaikan konfigurasi jika terdapat perubahan pada "nama database, username, password" atau yang lainnya
6. Jika melakukan perubahan nama folder directory utama (SI-PT), lakukan konfigurasi base Url pada file "baseUrl.php"
